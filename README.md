# BackJobsScreen
Created this project to do a backend to JobsScreen(https://github.com/NicolasSuzuki/JobsScreen)<br/>
Using NodeJs with knex, express and sqlite3

## Running
### Install the dependences 
`npm install`
### Run the project
`node src/index.js`
